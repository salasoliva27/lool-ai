# lool-ai

B2B SaaS virtual try-on widget for Mexican optical SMEs.

White-label widget embedded on store website or WhatsApp catalog. No 3D modeling — simple product photo upload. Priced in MXN. Onboarded in Spanish.

**Target market:** Independent optical stores in Roma, Condesa, Polanco, Lomas, and other CDMX SME-dense neighborhoods.

**Pricing:** ~800–1,500 MXN/month per store.

**Legal flag:** Handles facial image data → LFPDPPP compliance required before collecting real user data.
